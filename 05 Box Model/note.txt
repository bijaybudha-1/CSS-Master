----> Box Model

--> Margin
- Margin: Which is apply the outer surface (section) of the border.

--> Syntax:
margin-top: 10px;
margin-right: 10px;
margin-bottom: 10px;
margin-left: 10px;


--> shorthand for margin: 
- margin: 5px 10px 15px 20px; (top, right, bottom, left) (apply first value for top, second value for right, third value for bottom and another last value for left).

- margin: 10px 20px; (first value apply for top and bottm. another value is apply for left and right).

- margin: 10px 15px 20px; (first value for top, second value for right and left, another last (third) value for bottom).


----> Padding
--> Padding: padding is used or apply between the inner section (surface) of border and outer section (surface) of content.

Syntax:
- padding-top: 10px;
- padding-right: 10px;
- padding-bottom: 10px;
- padding-left: 10px;

--> shorthand for pading: 
- padding: 5px 10px 15px 20px; (top, right, bottom, left) (apply first value for top, second value for right, third value for bottom and another last value for left).

- padding: 10px 20px; (first value apply for top and bottm. another value is apply for left and right).

- padding: 10px 15px 20px; (first value for top, second value for right and left, another last (third) value for bottom).

- padding: Padding is apply between the content and borer (outer the content section and inner the border section).
